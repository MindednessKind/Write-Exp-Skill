#Crypto#

# RC4算法

> [!IMPORTANT]
>
> RC4是一种对每个字节进行**逐个加密**的数据流加密算法

# 加密过程

-
  ### 初始化S表 KSA
-
  ### 生成密钥流 PRGA
  > [!TIP]
  >
  > S表一旦完成初始化，种子密钥就不再被使用。

---

## 初始化S表 KSA

- Step1 对**S表**线性填充，一般为256字节

```python
#线性填充可记作：
def_length = 256 #可为其他值
s = list(range(def_length))
```

- Step2 使用**种子密钥**填充另一个与<u>S表大小相等</u>的**K**表

```python
seed = "" #假设已经拥有 seed ,大小为n
n = len(seed)

for i in range(def_length):
  k[i] = seed[i%n]
```

- Step3 用**K表**对**S表**进行**初始置换**

```python
#该初始置换为本次定义，其他RC4算法不一定为此式
j = 0
for i in range(def_length):
  j = ( j + s[i] + k[i] ) % def_length
  s[i],s[j] = s[j],s[i] #swap(s[i],s[j])
```

### 总体KSA算法封装：

```python
def KSA(seed,box_size):
    seed_len = len(seed)
    s_box = list(range(box_size))
    j=0
    
    for i in range(box_size):
        j = (j + s_box[i] + seed[i % len(seed)]) % box_size
        s_box[i], s_box[j]=s_box[j], s_box[i]
    return s_box
```

## 生成密钥流 PRGA

> ```
> 为每个待加密的字节生成一个伪随机数，用来异或。
> ```

**用已经完成初始化的S表生成一个伪随机数，用来异或（长度与明文长度相同）**

> **在种子代码和S表大小不变时，其伪随机值不变**

```python

#记明文长度为 plain_len
for i in range(plain_len):
  i = ( i + 1 ) % def_length
  j = ( j + s[i] ) % def_length
  s[i],s[j] = s[j],s[i] #swap(s[i],s[j])
  t = ( s[i] + s[j] ) % def_length
  key[i] = s[t] #key数组即存储了用来异或的伪随机数
```

### PRGA算法封装：

```python
def PRGA(s_box,box_size):
    i,j =0,0
    while True:
        i = (i+1) % box_size
        j = (j+1) % box_size
        s_box[i],s_box[j] = s_box[j],s_box[i]
        t = (s_box[i] + s_box[j])%box_size
        yield s_box[t]
```

# 最后对明文进行异或运算生成密文

```python
for i in range(plain_len):
  encrypt_data[i] = plain_txt[i] ^ key[i]
#encrypt_data即为加密后的密文
```

**注意：异或运算是可逆的，即可有**

> plain_text[i] = encrypt_data[i] ^ key[i]

### 对总体封装：

```python
def RC4(seed, plain_text, box_size:int = 256):
    #step 1   KSA
    s_box = KSA(seed,box_size)
    #step 2 PRGA
    progress = PRGA(s_box,box_size)
    encrypt_data = []
    for i in plain_text:
        encrypt_data.append( i ^ next(progress) )
    return bytes(encrypt_data)
```
