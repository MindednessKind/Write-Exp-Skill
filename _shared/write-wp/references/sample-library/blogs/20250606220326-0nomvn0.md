# 更改配置

我们这里使用 最新版Ubuntu的默认网络管理工具：Netplan

我们使用YAML文件编写其配置文件

首先，我们在 /etc/netplan 中新建一个名为 `01-network-manager-all.yaml` 的文件

```bash
sudo touch 01-network-manager-all.yaml
```

我们在其中写入

```yaml
network:
  version: 2
  renderer: NetworkManager #将默认的渲染器 systemd-networkd 改为 NetworkManager
  ethernets:
    ens33:
      dhcp4: no #设置为静态IP
      addresses: [192.168.192.128/24] #将特定静态IP设置好
      routes:
        - to: 0.0.0.0/0            # 默认路由
          via: 192.168.192.2        # 网关地址
      nameservers:
        addresses: [8.8.8.8, 8.8.4.4] #DNS设置
```

之后运行

```bash
sudo netplan try
```

如果没有报错，那么它会对你是否应用此设置进行一次询问。

我们这时使用 `ip a` 命令测试其是否成功更改

若静态IP已被应用，则这次的静态IP设置就大功告成了！
